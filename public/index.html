01

What is UTPT?

UTPT (Unified Training Performance Tracker) is a platform designed to track coding progress, GitHub activity, immersion performance, and overall student growth in one place.

02

Which coding platforms are supported?

UTPT currently supports integration with platforms like LeetCode, GitHub, Codeforces, and GeeksforGeeks for performance tracking and analytics.

03

Can trainers monitor entire batches?

Yes. Trainers can analyze batch-wise performance, identify top and low performers, monitor immersion scores, and track coding consistency.

04

How are rankings calculated?

Rankings are based on coding activity, problem-solving progress, GitHub contributions, immersion performance, streaks, and overall consistency.

05

Is the dashboard mobile-friendly?

No. but UTPT is fully responsive and optimized for desktops devices.

06

How often is student data updated?

Student coding and GitHub data are updated automatically using scheduled background sync systems and APIs.
